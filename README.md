**Credit Card Default Risk Prediction**

Built an end-to-end ML project to predict credit card default risk using the UCI Credit Card Default dataset (30K customer records). The goal was to help financial institutions identify high-risk customers and reduce potential losses.

🚀 **Highlights**
Performed EDA to analyze repayment behavior and default patterns
Applied feature engineering using payment ratio, average bill amount, and max delay
Compared Random Forest and XGBoost models for prediction
Used hyperparameter tuning with Stratified Cross Validation
Implemented SHAP for model explainability
Achieved ~85% Accuracy and ~0.87 ROC-AUC with tuned XGBoost

🛠 **Tech Stack**
Python, Pandas, NumPy, Scikit-learn, XGBoost, Matplotlib, Seaborn, SHAP

📁 **Dataset**

UCI Credit Card Default Dataset from Kaggle
